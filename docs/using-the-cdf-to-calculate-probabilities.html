Chapter 6 Using the CDF to Calculate Probabilities

Recall that the CDF can characterize a random variable, how is this done? Considered the events that are represented as the following inequalities, if \(P(X \le t)\) is known for all \(t \in \mathbb{R}\) all of the following events can be calculated

\[ (X \le a) = \qquad (X < a) \qquad (X \ge a) \qquad (X > a) \qquad (X = a) \]

\[ (a \le X \le b) \qquad (a < X < b) \qquad (a < X \le b) \qquad (a \le X < b) \]

6.1 Discrete Case

The CDF is defined as \(F(X) = P(X \le x)\)

Remember to use these to find the following probabilities:

Discrete Probabilities
\(P(X = x)=1 - P(X\le)\)
\(P(X > x)=1 - P(X\le a)\) Clearly this is the complimentary end of the CDF

\(P(X = x)=1 - P(X\le)\)

\(P(X < x)=1 - P(X\le)\)

\(P(X > x)=1 - P(X\le a)\)

\(P(X \ge x)=1 - P(X\le)\)

\(P(a < X \le b)= P(X\le a) - P(X\le a-)\)

\(P(a \le X < b)=1 - P(X\le)\)
